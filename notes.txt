Setup Nginx
- sudo apt update
- sudo apt-get install nginx -y
- sudo systemctl status nginx
- Tambahkan inbound rules untuk server nginx
  HTTP Anywhere dan HTTPS Anywhere

Config Nginx sebagai Reverse Proxy Server
- sudo vim /etc/nginx/sites-available/default
  location / {
    proxy_pass http://localhost:3000; # your app's port
    proxy_http_version 1.1;
    proxy_set_header Upgrade $http_upgrade;
    proxy_set_header Connection 'upgrade';
    proxy_set_header Host $host;
    proxy_cache_bypass $http_upgrade;
  }
- Simpan lalu sudo systemctl restart nginx
- Akses dengan Public IPv4 address

Terapkan Limit Access
- sudo vim /etc/nginx/sites-available/default
- Pada baris awal berkas tambahkan limit_req_zone $binary_remote_addr zone=one:10m rate=30r/m;
- Tambahkan dalam blok location / limit_req zone=one;
- limit_req zone=one;
- Jika diakses lebih dari 1 request / 2 detik maka error 503
- Hapus inbound traffic yang langsung mengarah ke aplikasi (Custom TCP) supaya jalur untuk mengakses aplikasi hanya tersedia melalui reverse proxy server
- sudo nginx -t && sudo nginx -s reload (ketika ada perubahan config)
